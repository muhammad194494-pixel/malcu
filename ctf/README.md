# 🚩 CTF Writeups

Koleksi writeup dan solve script dari kompetisi Capture The Flag.

## Structure

```
ctf/
├── 2026/                # Year
│   └── <ctf-name>/      # Competition
│       ├── README.md     # Overview & summary
│       ├── web/          # Web exploitation
│       ├── pwn/          # Binary exploitation
│       ├── crypto/       # Cryptography
│       ├── rev/          # Reverse engineering
│       ├── forensics/    # Forensics
│       └── misc/         # Miscellaneous
```

## Format Writeup

Setiap writeup harus mencakup:

1. 📝 **Challenge description**
2. 🔍 **Initial analysis**
3. ⚔️ **Exploit approach**
4. 🛠 **Solution script**
5. 🚩 **Flag**
