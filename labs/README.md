## Setup

Setiap lab akan memiliki `docker-compose.yml` dan `README.md` sendiri.

```bash
cd labs/<lab-name>
docker-compose up -d
```

## ⚠️ Warning

Lab ini mengandung kerentanan yang disengaja. **Jangan expose ke internet atau network publik!**
